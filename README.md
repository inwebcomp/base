# InWeb framework

## Requirements
- Laravel >=5.7

## Installation

1. Install package via *composer require*
    ```
    composer require inweb/base
    ```
    or add to your composer.json to **require** section and update your dependencies
    ```
    "inweb/base": "*"
    ```
2. Run migrations
    ```
    php artisan migrate
    ```

You are ready to go!
